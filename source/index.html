TavernBoard Cloud

Infraestrutura cloud para um fórum online de RPG

Sobre o projeto

Este ambiente simula uma infraestrutura Linux containerizada para hospedagem de um fórum online de RPG, utilizando Docker, Apache e Shell Script.

O objetivo é automatizar rotinas de operação como deploy, backup, monitoramento, permissões, logs e geração de relatórios.

Módulos simulados

Campanhas

Organização de aventuras e mesas de RPG.

Personagens

Cadastro e gerenciamento dos personagens dos jogadores.

Fórum

Discussões, tópicos e comentários da comunidade.

Tecnologias utilizadas

Ubuntu Docker Apache Shell Script Cloud Computing
